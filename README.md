# cryptohashpy
Commonly used hash functions

## Installation

Use the setup.py script to install this C extension.

```bash
python setup.py install
```

## Usage

```python
import cryptohashpy

cryptohashpy.md5(b'word') # returns 'C47D187067C6CF953245F128B5FDE62A'
```
